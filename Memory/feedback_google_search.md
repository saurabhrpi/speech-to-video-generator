---
name: Google search when uncertain
description: Use web search proactively when unsure about direction, answers, or approaches
type: feedback
---

When in doubt about which direction to think, what to answer, or what approach to take — do a Google search rather than guessing or hedging.

**Why:** The user wants well-informed, grounded responses. Searching is better than speculating.

**How to apply:** Proactively use WebSearch whenever uncertain about technical approaches, model capabilities, API behavior, best practices, or any factual question. Don't treat searching as a last resort — treat it as a first instinct when confidence is low.
