Design system interview rounds must match the depth of previous sessions: each topic gets its own round with multiple questions presented as a "carousel" of cards with choices. Don't bundle 3 topics into one shallow round.

**Why:** User called out lazy effort when I condensed remaining topics into single questions each. Previous session had rich multi-question exploration per topic.

**How to apply:** Each topic = its own round. Each round = multiple questions exploring different facets. Present as numbered cards with lettered choices. Go deep, not wide.
